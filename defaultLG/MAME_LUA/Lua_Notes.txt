You need to have the Lua scripts to then output the data for the light gun force feedback. Usually the Lua scripts go into the 'scripts' directory in MAME. 

Option 1 (one off):
To load the lua script, please see the example below.
mame.exe area51 -autoboot_script "D:\MAME\scripts\area51.lua"

Option 2 (permanent):
In the 'ini' directory in MAME, create an INI file with the ROM's name. Area 51, would be 'area51.ini' file. Then boot the info below into the INI file.
autoboot_script "scripts/area51.lua"

Or Use with Full Path
autoboot_script "D:\Path\To\MAME\scripts\area51.lua"


To change the recoil or damage pulse duration, change these values in the lua script.
RECOIL_PULSE_DURATION_MS 	: Duration (in Milliseconds) of the "recoil" signal for your solenoid/motor
DAMAGED_PULSE_DURATION_MS	: Duration (in Milliseconds) of the "damaged" signal for your motor/light/whatever
